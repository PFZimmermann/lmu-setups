Setup fixo (fixed setup)

Os controles de tração usam o padrão atual do LMU: TC, TC Power Cut e TC Slip. A tabela mostra apenas os controles disponíveis para o carro selecionado.

Como usar arquivos .svm

  1. Clique em um botão de download para baixar um placeholder .svm com instruções.
  2. Substitua pelo seu setup real exportado do Le Mans Ultimate.
  3. setups/[classe]/[carro]/[pista]/(qualifying|race|sprint).svm
  4. Atualize os links no js/app.js para apontar para seus arquivos reais.
